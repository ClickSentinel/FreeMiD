# Security Policy

## Reporting a vulnerability

If you believe you have found a security issue in FreeMiD, do not open a public issue.

Please report it privately to the project maintainers through the repository owner contact channel.

Include:

- A short description of the issue
- Steps to reproduce
- Any affected versions
- Any logs, screenshots, or proof of concept you can safely share

## What we will do

We will review reports as soon as practical and respond with next steps. If the issue is valid, we will work on a fix before public disclosure.
